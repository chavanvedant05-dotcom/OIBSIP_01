# Simple Calculator

A basic calculator project made using HTML, CSS, and JavaScript.

## 📌 Features
- Addition
- Subtraction
- Multiplication
- Division
- Percentage
- Clear Display
- Simple UI Design

---

## 📁 Files

### 1. index.html
Contains the structure of the calculator.

### 2. style.css
Contains the styling and layout design.

### 3. script.js
Contains the calculator functionality.

---

## 🚀 Technologies Used
- HTML5
- CSS3
- JavaScript

---

## ▶️ How to Run
1. Download all files.
2. Keep all files in one folder.
3. Open `index.html` in your browser.
